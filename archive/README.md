# Historical pre-CBOR archive

This directory preserves 52 superseded AIKDNA release entries:

- `index.json` — machine-readable archive index;
- `artifacts/` — 47 byte-preserved historical artifacts tracked by the repository;
- `checksums/` — 47 checked-in historical checksum sidecars;
- `showcase/` — dated descriptive pages;
- `evidence/` — retained historical compatibility/audit records;
- GitHub Releases — original downloadable bytes and release-side checksums.

The archive is not current onboarding. The current runtime intentionally
rejects these files. Archive pages may provide download and integrity metadata,
but must not provide current validation, planning, installation, or Agent
loading instructions.

The other five index entries remain preserved through their GitHub Releases;
they never had repository-tracked artifact/sidecar files. Historical artifacts,
checksums, release tags, URLs, pages, and experiments are preserved.
No archive binary may be regenerated, directly decoded, renamed, or relabeled
as current CBOR.
