# Evidence Directory

This directory contains public-safe evidence manifests for published KDNA assets
in the open/kdna-assets repository.

## Contents

- `manifest-v1.json` — Evidence manifest for all published assets
- `compatibility.json` — Cross-asset compatibility matrix

## What Is NOT Here

Per the public evidence policy:
- No private benchmark data
- No customer-specific evidence
- No unpublished experimental results
- No internal evaluation metrics
- No creator/studio internal IDs

## Generation

Evidence manifests are generated by `kdna asset-evidence` in the kdna-cli.
Re-run after any asset content change.
