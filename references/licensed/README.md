# Licensed reference asset entries

There are currently no licensed assets in this public display.

A future licensed reference owns one directory and its own license:

```text
references/licensed/<asset-slug>/
  <asset-slug>-<version>.kdna
  <asset-slug>-<version>.kdna.sha256
  LICENSE
  README.md
```

The root repository license does not license an entry by implication. A
licensed entry must declare its own license and authorization terms in
`index/current.json`.
